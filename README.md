# ML_BASIC_LOGISTIC_REGRESSION
# 📊 Logistic Regression Classification — Student Pass Prediction

## 📌 Project Overview

This project demonstrates how to implement **Logistic Regression using scikit-learn** to predict whether a student will **pass or fail** based on the number of study hours.

It follows a standard machine learning workflow:

* Load dataset
* Split into training and testing sets
* Train logistic regression model
* Evaluate performance
* Predict probabilities
* Visualize sigmoid curve

This project is designed for beginners learning **classification in machine learning** using Python.

---

## 🎯 Problem Statement

Given the number of study hours, predict:

* **1 → Pass**
* **0 → Fa
